# Data-cleaning

## Super Dirty Students Data Cleaning Project

This Python project is designed to clean and normalize a messy student dataset (`super_dirty_students.csv`) and export a fully cleaned version (`super_dirty_students_cleaned.csv`). It handles string cleaning, numeric conversion, date parsing, JSON extraction, email and phone validation, address parsing, duplicate removal, and QA checks.

---

## Project Overview

The dataset contains messy student information with inconsistent formatting, missing values, and nested data. This project standardizes all columns, ensures correct data types, and prepares the dataset for analysis or machine learning tasks.

**Key Tasks Performed:**
- Data exploration and understanding
- String, numeric, and date cleaning
- Email and phone validation
- JSON parsing and flattening
- Address parsing
- Duplicate and missing data handling
- Data normalization
- QA checks to ensure integrity
